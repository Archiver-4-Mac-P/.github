## 📋 Overview

<a href="#archiver4">Archiver 4</a> is a native Mac archive manager that handles the full lifecycle of compressed files — creating, opening, browsing, extracting, splitting, and encrypting archives across 30-plus formats through a clean macOS interface with capabilities that go well beyond the built-in Archive Utility.

The <a href="#archiver4">format support breadth</a> covers both extracting and creating: ZIP, RAR, 7-Zip, TAR, GZip, BZip2, XZ, ISO, DMG, and many more open without format errors, while Archiver 4 creates ZIP and 7-Zip archives for output. Any compressed file received from any source — developer project archives, downloaded software packages, shared asset collections — opens in Archiver 4 without the "unsupported format" problem that macOS's built-in Archive Utility frequently hits with non-ZIP formats. The <a href="#archiver4">preview without extraction</a> is one of Archiver 4's most practical features — opening any archive as a browsable file view that shows the complete contents, file sizes, and folder structure without extracting anything. Individual files open directly from inside the archive, and only the needed files extract rather than the entire archive.

The <a href="#archiver4">password-protected archive creation</a> builds AES-256 encrypted ZIP and 7-Zip archives for sensitive file sharing — the strongest encryption that ZIP and 7-Zip support, in a standard format that any recipient opens with their own unarchiving tool using the correct password. The <a href="#archiver4">archive splitting capability</a> divides large archives into multiple parts of a specified size — for email attachment limits, cloud storage size restrictions, or any context where the complete archive exceeds a size threshold. Split parts rejoin through Archiver 4's merge function when all parts are available.

<a href="#archiver4">Secret Safe encrypted vault</a> creates a local encrypted container for sensitive files — a password-protected storage area within Archiver 4 that holds documents, credentials, and private files accessible only with the correct password, providing secure local storage beyond standard archive formats. <a href="#archiver4">Drag-and-drop archive creation</a> builds archives by dropping files and folders onto Archiver 4's window — no menu navigation, no format dialog complexity, configure the format and options and proceed.

<a href="#archiver4">Batch archive operations</a> process multiple archives simultaneously — queuing several archives for extraction in one operation rather than handling each individually. <a href="#archiver4">Finder context menu and Quick Look integration</a> adds Archiver 4 operations to the Finder right-click menu and enables archive content preview through the spacebar Quick Look shortcut without opening Archiver 4 fully.
